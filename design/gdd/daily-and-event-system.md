# 日常与活动系统设计 - 《神界：混沌边境》

## 1. 日常系统

### 1.1 每日任务
| 任务类型 | 任务内容 | 奖励 |
|----------|----------|------|
| 登录 | 每日登录游戏 | 星辉×100 |
| 消耗体力 | 消耗100体力 | 金币×5000 |
| 完成副本 | 完成3次任意副本 | 经验书×3 |
| 角色培养 | 进行1次角色升级或突破 | 强化石×2 |
| 抽卡 | 进行1次抽卡 | 星辉×50 |
| 羁绊互动 | 与角色互动1次 | 好感礼物×1 |
| 公会任务 | 完成1次公会任务 | 公会币×100 |

### 1.2 每周任务
| 任务内容 | 奖励 |
|----------|------|
| 累计登录7天 | 星辉×500 |
| 完成每日任务×35 | SSR角色碎片×10 |
| 消耗体力×1000 | 金币×50000 |
| 完成深渊塔×10层 | 技能书×5 |
| 公会贡献×500 | 公会币×1000 |

### 1.3 体力系统
| 等级 | 体力上限 | 恢复速度 |
|------|----------|----------|
| 1-30 | 100 | 1点/6分钟 |
| 31-60 | 120 | 1点/5分钟 |
| 61-100 | 150 | 1点/4分钟 |

体力购买：
- 每日前2次：50星晶石/次
- 第3-4次：100星晶石/次
- 第5-6次：150星晶石/次

## 2. 副本系统

### 2.1 主线关卡
```
章节结构：
├── 第1章：觉醒之始（10关）
├── 第2章：边境之旅（15关）
├── 第3章：暗影初现（15关）
├── ...
└── 第10章：神界终焉（20关）

每关消耗：6体力
首通奖励：星辉×50 + 突破材料
重复奖励：金币 + 经验 + 随机材料
```

### 2.2 材料副本（每日轮换）
| 周几 | 副本类型 | 主要产出 |
|------|----------|----------|
| 周一/四 | 经验试炼 | 经验书 |
| 周二/五 | 金币试炼 | 金币 |
| 周三/六 | 技能试炼 | 技能书 |
| 周日 | 装备试炼 | 装备、强化石 |
| 每日 | 元素试炼（轮换） | 对应属性突破材料 |

### 2.3 深渊塔
- **层数**: 100层
- **重置周期**: 每月1日
- **规则**: 
  - 每层1场BOSS战
  - 角色阵亡后无法复活
  - 每通关10层可领取奖励
- **奖励**:
  - 每10层: 稀有材料
  - 每50层: SSR角色碎片
  - 通关100层: 专属称号 + SSR装备

### 2.4 世界BOSS
- **开放时间**: 每周六、日
- **参与次数**: 每日3次
- **BOSS列表**:
  - 烈焰魔龙（火属性）
  - 冰霜巨人（冰属性）
  - 雷霆神兽（雷属性）
  - 暗影领主（暗属性）
  - 圣光守护者（光属性）
- **奖励**: 根据伤害排名发放

## 3. 活动系统

### 3.1 活动类型

#### 限时活动
| 活动名称 | 周期 | 内容 |
|----------|------|------|
| **降临活动** | 2周 | 限定角色UP池 + 专属剧情 |
| **节日活动** | 1-2周 | 节日主题剧情 + 限定奖励 |
| **联动活动** | 3周 | 联动角色 + 联动剧情 |
| **周年庆典** | 1个月 | 全民福利 + 特别活动 |

#### 常驻活动
| 活动名称 | 开放时间 | 内容 |
|----------|----------|------|
| **竞技场赛季** | 每月 | PVP排位赛 |
| **公会战** | 每双周 | 公会VS公会 |
| **材料狂欢** | 周末 | 材料副本掉落×2 |
| **抽卡祭** | 月末 | 抽卡返利活动 |

### 3.2 活动货币系统
每种活动有专属货币：
- 降临币 → 兑换限定角色碎片
- 节日券 → 兑换节日限定道具
- 公会战勋章 → 兑换公会商店物品

### 3.3 活动结构模板

```
活动周期（14天）:
├── Day 1-3: 剧情解锁
├── Day 4-10: 核心玩法开放
├── Day 11-13: 材料兑换期
└── Day 14: 最终BOSS + 结算

活动奖励:
├── 免费线: 活动角色×1 + 基础材料
├── 通行证线: 额外角色碎片 + 稀有材料
└── 氪金线: 专属皮肤 + 专属武器
```

## 4. 竞技场系统

### 4.1 段位系统
| 段位 | 分数区间 | 奖励 |
|------|----------|------|
| 青铜 | 0-999 | 星辉×100 |
| 白银 | 1000-1499 | 星辉×200 |
| 黄金 | 1500-1999 | 星辉×300 + SR碎片 |
| 铂金 | 2000-2499 | 星辉×500 + SSR碎片 |
| 钻石 | 2500-2999 | 星辉×700 + SSR碎片×2 |
| 大师 | 3000-3499 | 星辉×1000 + 专属称号 |
| 王者 | 3500+ | 星辉×1500 + 限定皮肤 |

### 4.2 匹配规则
- 系统匹配相近段位玩家
- 可选择挑战NPC（分数较少）
- 每日免费挑战5次
- 额外挑战次数可购买

### 4.3 防守阵容
- 玩家设置防守队伍
- 被挑战失败不掉分
- 成功防守获得额外奖励

## 5. 公会系统

### 5.1 公会基础
- **创建条件**: Lv.20 + 500星晶石
- **人数上限**: 50人
- **公会等级**: 1-10级
- **公会福利**:
  - 公会商店
  - 公会技能（永久属性加成）
  - 公会BOSS

### 5.2 公会战
- **周期**: 每双周
- **形式**: 公会VS公会
- **规则**:
  - 争夺据点
  - 攻防战
  - 最终积分排名
- **奖励**:
  - 公会币
  - 公会限定角色
  - 公会专属皮肤

### 5.3 公会BOSS
- **开放**: 每周三、六
- **形式**: 全公会成员共同挑战
- **奖励**: 公会币 + 随机稀有材料

## 7. 成就系统

### 7.1 成就分类
| 类别 | 示例 | 奖励 |
|------|------|------|
| 角色 | 收集100个角色 | 星辉×2000 |
| 战斗 | 累计胜利1000场 | 称号 |
| 剧情 | 通关全部主线 | SSR装备 |
| 社交 | 好友数达到50 | 好感礼物 |
| 收集 | 收集全角色立绘 | 专属皮肤 |

### 7.2 隐藏成就
- 特殊条件触发
- 更丰厚的奖励
- 稀有称号

## 8. 商店系统

### 8.1 商店类型
| 商店 | 货币 | 主要商品 |
|------|------|----------|
| 星辉商店 | 星辉 | 角色/武器碎片 |
| 公会商店 | 公会币 | 公会限定道具 |
| 竞技场商店 | 竞技场币 | PVP装备 |
| 活动商店 | 活动币 | 限时角色/道具 |
| 好感商店 | 好感币 | 角色礼物 |

### 8.2 刷新机制
- 每周刷新: 星辉商店
- 每月刷新: 公会商店、竞技场商店
- 活动期间: 活动商店

---

*设计理念：让日常成为习惯，让活动成为期待*
