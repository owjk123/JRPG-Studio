# 养成系统设计 - 《神界：混沌边境》

## 1. 系统概述

养成系统是让玩家与角色建立深厚羁绊的核心，通过多维度的成长路线，让每个角色都能成为玩家独一无二的存在。

## 2. 等级系统

### 2.1 等级上限
- **基础上限**: Lv.100
- **突破加成**: 每次突破+20级，最高Lv.200（5次突破）

### 2.2 经验获取
| 来源 | 经验值 | 限制 |
|------|--------|------|
| 主线关卡 | 基础×关卡系数 | 无限 |
| 经验副本 | 高额经验 | 每日3次 |
| 日常任务 | 固定经验 | 每日刷新 |
| 经验道具 | 直接使用 | 无限 |

### 2.3 经验曲线
```
升级所需经验 = 基础经验 × 等级^1.5
```

### 2.4 等级奖励
- 每10级: 属性提升+解锁技能槽
- 每20级: 外观变化（可选）
- 每50级: 专属称号

## 3. 突破系统

### 3.1 突破阶段
| 阶段 | 等级上限 | 材料需求 | 特殊奖励 |
|------|----------|----------|----------|
| 0阶 | Lv.100 | - | - |
| 1阶 | Lv.120 | 角色碎片×30 | 解锁被动技能槽 |
| 2阶 | Lv.140 | 角色碎片×50 + 元素结晶 | 属性飞跃+10% |
| 3阶 | Lv.160 | 角色碎片×80 + 稀有材料 | 解锁终极技能强化 |
| 4阶 | Lv.180 | 角色碎片×120 + 传说材料 | 外观变化 |
| 5阶 | Lv.200 | 角色碎片×200 + 神器碎片 | 全属性+20% |

### 3.2 突破材料获取
- **角色碎片**: 抽卡重复转化、商店兑换、活动奖励
- **元素结晶**: 对应属性副本
- **稀有材料**: 高难度副本、世界BOSS
- **传说材料**: 活动限定、深渊塔

## 4. 技能系统

### 4.1 技能分类
| 类型 | 数量 | 解锁条件 |
|------|------|----------|
| 普通攻击 | 1个 | 初始 |
| 主动技能 | 3个 | Lv.1/20/50 |
| 被动技能 | 3个 | 突破1阶解锁槽位 |
| 终极技能 | 1个 | Lv.30 |
| 种族天赋 | 1个 | 初始 |

### 4.2 技能升级
- **升级消耗**: 技能书 + 金币
- **等级上限**: Lv.15
- **效果提升**: 每级伤害/效果+5-10%

### 4.3 技能树（高级角色专属）
```
主动技能分支:
├── 威力强化路线: 伤害+20%，消耗+10MP
├── 效率强化路线: 消耗-20%，伤害-10%
└── 效果强化路线: 附加状态效果

被动技能分支:
├── 触发强化: 触发概率+15%
├── 效果强化: 效果数值+25%
└── 持续强化: 持续回合+2
```

## 5. 装备系统

### 5.1 装备槽位
| 槽位 | 主要属性 | 次要属性 |
|------|----------|----------|
| 武器 | 攻击力 | 暴击/命中 |
| 头盔 | 防御力 | HP/抵抗 |
| 铠甲 | HP | 防御/速度 |
| 护手 | 攻击/速度 | 暴击伤害 |
| 饰品 | 特殊属性 | 随机属性 |
| 专属武器 | 全属性 | 技能强化 |

### 5.2 装备品质
| 品质 | 主属性倍率 | 副属性数量 | 获取途径 |
|------|------------|------------|----------|
| 普通(N) | ×1.0 | 0-1 | 普通关卡 |
| 稀有(R) | ×1.2 | 1-2 | 精英关卡 |
| 史诗(SR) | ×1.5 | 2-3 | BOSS关卡 |
| 传说(SSR) | ×1.8 | 3-4 | 活动限定 |
| 神器(UR) | ×2.2 | 4 + 套装效果 | 抽卡/活动 |

### 5.3 装备强化
- **强化上限**: +20
- **强化消耗**: 强化石 + 金币
- **强化效果**: 主属性每级+5%
- **特殊强化**: +10/15/20解锁额外属性

### 5.4 套装系统
| 套装名 | 2件效果 | 4件效果 |
|--------|--------|--------|
| 烈焰之心 | 攻击+15% | 火属性伤害+30% |
| 冰霜守护 | 防御+20% | 受击时概率冻结敌人 |
| 疾风之翼 | 速度+15 | 行动后30%概率再动 |
| 不朽之魂 | HP+25% | 死亡时复活一次(50%HP) |
| 虚空之力 | 暴击+10% | 暴击伤害+50% |

## 6. 羁绊系统

### 6.1 羁绊等级
| 等级 | 所需好感度 | 奖励 |
|------|------------|------|
| Lv.1 | 0 | 角色档案解锁 |
| Lv.2 | 100 | 属性+2%，日常语音解锁 |
| Lv.3 | 300 | 属性+5%，战斗语音解锁 |
| Lv.4 | 600 | 属性+8%，羁绊剧情解锁 |
| Lv.5 | 1000 | 属性+12%，专属立绘解锁 |
| Lv.6-10 | 累计递增 | 属性递增+专属奖励 |

### 6.2 好感度获取
| 方式 | 好感度 | 限制 |
|------|--------|------|
| 赠送礼物 | 10-100 | 无限 |
| 一起战斗 | 5/场 | 每日10次 |
| 羁绊剧情 | 50/章 | 解锁后 |
| 日常互动 | 10/次 | 每日1次 |

### 6.3 礼物系统
| 礼物类型 | 好感度 | 喜好加成 |
|----------|--------|----------|
| 普通礼物 | +10 | - |
| 稀有礼物 | +30 | 喜好×1.5 |
| 专属礼物 | +100 | 喜好×2.0 |
| 生日礼物 | +200 | 固定倍率 |

### 6.4 羁绊技能
- **解锁条件**: 羁绊Lv.5
- **效果类型**: 
  - 战斗中被动增益
  - 特殊合体技能
  - 队伍光环效果

## 10. 养成节奏规划

### 10.1 新角色培养路线
```
Day 1: 升级至Lv.50，解锁全部技能
Day 2-3: 升级至Lv.80，装备基础套装
Day 4-7: 突破1阶，技能升级
Week 2: 突破2阶，套装完善
Week 3-4: 突破3阶，专属武器获取
长期: 突破5阶，命座解锁
```

### 10.2 资源分配建议
- **优先级**: 主力角色 > 辅助角色 > 收藏角色
- **资源倾斜**: 集中资源培养3-5个核心角色
- **平衡发展**: 保持角色池有一定广度

---

*设计理念：让养成成为一段旅程，而不是单纯的数值堆砌*
