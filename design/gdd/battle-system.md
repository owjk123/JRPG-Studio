## 战斗核心机制

### ATB半即时战斗系统

**核心理念**：保留回合制的策略深度，融入即时反应的操作感。

```
┌─────────────────────────────────────────────────────────────┐
│                     ATB 时间槽系统                           │
├─────────────────────────────────────────────────────────────┤
│  角色1 [████████░░░░░░░░░░] ████░░░░░░░░░░░░░░░░░░░░░░░░░░  │
│  角色2 [██████░░░░░░░░░░░] ██████░░░░░░░░░░░░░░░░░░░░░░░░  │
│  敌人A [██████████████░░░░] ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  │
│  角色3 [██░░░░░░░░░░░░░░░] ████░░░░░░░░░░░░░░░░░░░░░░░░░░  │
└─────────────────────────────────────────────────────────────┘
        ↑ ATB槽满了即可行动               ↑ 当前行动者
```

### 行动点（AP）系统

| 行动类型 | AP消耗 | 说明 |
|----------|--------|------|
| 攻击 | 3 AP | 普通攻击 |
| 技能 | 5-15 AP | 消耗根据技能威力 |
| 防御 | 2 AP | 减少50%伤害，回合结束获得少量AP |
| 物品 | 4 AP | 使用道具 |
| 移动/换位 | 2 AP | 在战场中移动 |
| 终极技能 | 20 AP | 消耗全部AP，释放华丽演出 |

### 速度与行动顺序

```
行动顺序计算公式：
行动权重 = 角色基础速度 × ATB充能系数 × 状态加成

速度等级：
S级: 200+  (天使族光翼加持)
A级: 150-199
B级: 100-149
C级: 50-99
D级: 1-49
```

### 连携系统（Chain Combo）

**触发条件**：同一回合内，连续使用同属性或同类型的技能

| 连携等级 | 条件 | 伤害加成 | 额外效果 |
|----------|------|----------|----------|
| Lv1 一连 | 2次同属性 | +15% | 音效强化 |
| Lv2 二连 | 3次同属性 | +35% | 屏幕震动 |
| Lv3 三连 | 4次同属性 | +60% | 伤害范围扩大 |
| Lv4 四连 | 5次同属性 | +100% | 附带元素异常 |
| Lv5 五连 | 6次同属性 | +150% | 终结技解锁 |

**终结技示例**：
- 火属性×5 → **炽焰爆轰** (全体火焰伤害+灼烧)
- 圣光×5 → **神圣制裁** (全体光伤害+眩晕)
- 暗影×5 → **深渊吞噬** (单体大伤害+即死判定)

## 伤害公式框架

### 基础伤害公式

```
最终伤害 = 基础伤害 × 属性克制 × 种族克制 × 暴击倍率 × 随机浮动
```

### 物理伤害公式

```
物理伤害 = (攻击方ATK - 防御方DEF/2) × 技能倍率 × 属性克制 × (1 + 伤害加成% - 伤害减免%)
```

### 魔法伤害公式

```
魔法伤害 = (攻击方MAG - 防御方RES/2) × 技能倍率 × 属性克制 × (1 + 伤害加成% - 伤害减免%)
```

### 暴击系统

```
暴击率 = 基础暴击率 + 装备加成 + 技能加成 + 状态加成
暴击伤害 = 普通伤害 × 暴击倍率(默认150%)

暴击判定流程：
1. 计算暴击率
2. 随机数 < 暴击率 → 触发暴击
3. 暴击伤害 = 普通伤害 × 暴击倍率
```

### 元素反应伤害

| 反应类型 | 触发条件 | 额外伤害 | 效果 |
|----------|----------|----------|------|
| **蒸发** | 火+水 | ×1.5 | 伤害提升50% |
| **融化** | 火+冰 | ×1.8 | 伤害提升80% |
| **超导** | 雷+冰 | ×1.5 | 防御-30%持续2回合 |
| **燃烧** | 火+风 | ×1.3 | 附加灼烧 |
| **感电** | 雷+水 | ×1.3 | 麻痹概率翻倍 |
| **冻结** | 水+冰 | 停止行动 | 冻结2回合 |

### 伤害减免机制

```
最终伤害减免 = 1 - (护盾减免 + 防御减免 + 属性抗性 + 状态减免)

护盾优先级：
1. 无敌状态（完全免疫）
2. 魔法/物理护盾（抵消1次攻击）
3. 防御姿态（-50%）
4. 属性抗性（根据元素）
5. 种族抗性（根据种族克制）
```
